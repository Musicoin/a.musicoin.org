Musicoin Wallet Applications

Manage your account(s) wisely

Musicoin Wallet (v0.5) is designed with simplicity in mind in order to provide the necessary functions to manage your account(s) and interact with the money system.

Please download the version corresponding to your Operating system from below:

Windows

32 Bit, 64 Bit
Decompress the package and run with Musicoin-client.exe file in the root folder

Mac

OSX
Decompress the package and run the .dmg file in the root folder

Linux

32 Bit, 64 Bit
Decompress the package and run "./Musicoin-client" in the root folder

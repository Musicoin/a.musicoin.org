Musicoin Wallet Applications

Manage your account(s) wisely

Musicoin Wallet is designed for simplicity but with necessary functions to manage your account(s) to interact with the money system. Please choose your versions to download:

Windows

32 Bit, 64 Bit
Decompress the package and run with Musicoin-client.exe file in the root folder

Mac

OSX
Decompress the package and run with .dmg file in the root folder

Linux

32 Bit, 64 Bit
Decompress the package and run "./Musicoin-client" in the root folder
